//This code was tested on MATLAB R2016a

1. Place all files in one folder:
    a. main.m
    b. VortexPanelMethod.m
    c. boundarylayer.m
    d. thermal.m
    e. naca1408.txt

2. The .txt file (Airfoil coordinates) follows selig format
   (Start from trailing edge and goes CCW)

3. Run main.m on the console 

4. Input known values. Example:

     Freestream velocity: 15 m/s
     Angle of Attack: 2 deg
     Freestream temperature: 300 K
     Airfoil temperature: 330 K
